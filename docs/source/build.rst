How To Build App
===================================
.. _buildpackage:

Build
------------

Jalankan perintah ini pada terminal :

.. code-block:: console

    mvn install -f pom.xml

Setelah selesai cek folder :

.. code-block:: console

    target/app

hasil generate sudah bisa dibuat zip untuk update offline & online
